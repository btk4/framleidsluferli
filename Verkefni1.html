Forritun og uppsetning vefsíðunnar

Uppbygging vefsíðunnar var fundin á HTML5UP!
Þaðan sótti ég Templateið Telephasic.

Ég hef unnið með html áður en ákvað samt að nýta mér þetta template þar sem mér leiðist að nota CSS.
Markmið vefsíðunnar er að innihalda öll verkefni sem ég geri í áfanganum Framleiðsluferli í HÍ ásamt því að sýna ferilskránna mína og aðra hluti sem mér dettur í hug.
Html kóðinn sjálfur var unninn í Brackerts hugbúnaðinum.
Brackets hugbúnaðurinn býður upp á forskoðun á vefsíðunni sem verið er að vinna í með aðstoð Google Chrome vafrans.
Vefsíðunni var hlaðið upp á GitHub. Ég var með aðgang þar fyrir á háskóla netfangi mínu.

Þar næst náði ég í Git hugbúnaðinn sem ég nota til að halda utan um breytingar á kóðanum þar sem þessi síða er ekki full kláruð. Allavega ekki á meðan þessu verkefni stóð.

Þegar ég var orðinn sáttur með stöðu vefsíðunnar þá fann ég möppuna þar sem ég geymi alla kóða vefsíðunnar, hægrismelli músinni og smelli á "Git Bash Here".
Við það ætti að poppa upp Terminal gluggi. Inn í þann glugga setjum við afrit af kóða sem gefinn er upp af GitHub þegar búið er til nýtt repository.

Git bað mig um að skrá mig inn á GitHub, eftir það þurfti ég bara að endurhlaða GitHub síðunni, við það birtust allar skrár sem ég hlóð þar inn á í gegnum Git.
Eftir það þurfti ég að smella á settings, finna þar GitHub Pages og breyta source úr "none" í "main" og vista.
Eftir þetta ætti síðan að vera komin á netið.
